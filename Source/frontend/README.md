# Frontend

Клиентская часть приложения Toy Shelf построена на React и Vite.

## Запуск

```powershell
npm install
npm run dev
```

## Основные возможности

- HTML-разметка через JSX.
- CSS-оформление интерфейса.
- Загрузка данных через Fetch API.
- Сохранение демо-сессии в LocalStorage.
- Сохранение корзины в LocalStorage.
- Начальная страница входа и регистрации.
- Регистрация пользователя с паролем.
- Вход пользователя по email и паролю.
- Вход администратора с логином `admin` и паролем `admin123`.
- Удаление пользователей администратором.
- Каталог товаров с картинками, категориями, ценами и остатками.
- Корзина с изменением количества и расчетом итоговой суммы.
- Оформление заказа с обновлением остатков на сервере.
